Hello Plugin — Extension API example

This entire page is a plugin, loaded via window.RAX_PLUGINS and RaxPluginLoader — no framework file was edited to make it work. Its source is examples/hello-plugin/index.js, and it uses exactly three registration calls:

  • RaxRegistry.registerWidget() — the card below
  • RaxRegistry.registerCommand() — open the command palette (Ctrl/Cmd+K) and run "Say Hello"
  • RaxRegistry.registerPage() — this page itself
